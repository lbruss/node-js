# Node.js

Este repositório documenta meu início no aprendizado de **JavaScript** utilizando **Node.js**.
Iniciante na área, então este repositório reune projetos simples, testes e pequenas práticas.

## Objetivo

O principal objetivo deste projeto é:

* Praticar conceitos básicos de JavaScript
* Entender o funcionamento do Node.js no ambiente de execução
* Desenvolver minha lógica de programação
* Construir uma base sólida para evoluir gradualmente

Este é um projeto focado em aprendizado.
O código pode não seguir padrões avançados ou boas práticas mais refinadas, pois estou em processo de desenvolvimento e aprendizado contínuo.
A ideia é evoluir com o tempo, melhorando tanto a organização quanto a qualidade do código.
